# 🌴 نادي النخيل — طريقة التشغيل

هذه حزمة جاهزة لتشغيل النظام على جهازك فعلياً. تحتاج **خطوة تجهيز واحدة** ثم أمر واحد للتشغيل.

---

## الخطوة 1: تثبيت Node.js (مرة واحدة على الجهاز)

النظام يحتاج بيئة **Node.js** لتشغيله.

- اذهب إلى **nodejs.org** وحمّل النسخة الموصى بها (LTS)
- ثبّتها بالضغط "التالي" حتى النهاية
- للتأكد أنها رُكّبت: افتح **موجّه الأوامر** (Command Prompt على ويندوز، أو Terminal على ماك) واكتب:
  ```
  node -v
  ```
  إن ظهر رقم إصدار (مثل `v20.x.x`) فأنت جاهز.

---

## الخطوة 2: تشغيل النظام

1. **فك ضغط** هذه الحزمة في مجلد على جهازك (مثل سطح المكتب)
2. افتح **موجّه الأوامر** داخل مجلد المشروع:
   - **ويندوز**: افتح المجلد، اكتب `cmd` في شريط العنوان أعلى النافذة واضغط Enter
   - **ماك**: افتح Terminal، اكتب `cd ` ثم اسحب المجلد إلى النافذة واضغط Enter
3. نفّذ الأمرين التاليين (الأول مرة واحدة فقط، والثاني كل مرة تشغّل فيها):

   ```
   npm install
   ```
   ```
   npm run dev
   ```

4. سيظهر رابط مثل:
   ```
   ➜  Local:   http://localhost:5173/
   ```
   افتحه في المتصفح (Chrome مفضّل) — **النظام يعمل الآن!** ✅

> سجّل الدخول باختيار أي مستخدم (المدير له كل الصلاحيات) — أي كلمة مرور تعمل في وضع العرض.

---

## 📱 فتح النظام من جهاز لوحي (على نفس شبكة الواي فاي)

عند تشغيل `npm run dev` سيظهر أيضاً رابط **Network** مثل:
```
➜  Network: http://192.168.1.15:5173/
```
افتح هذا الرابط في متصفح الجهاز اللوحي (المتصل بنفس الواي فاي) — يعمل النظام كاملاً باللمس.

> إن لم يظهر، تأكد أن الحاسوب والجهاز اللوحي على نفس الشبكة، وأن جدار الحماية لا يمنع المنفذ 5173.

---

## 💾 عن حفظ البيانات

- **يعمل فوراً**: بياناتك تُحفظ تلقائياً في متصفح الجهاز (تنجو من الإغلاق والتحديث)
- **للمزامنة بين عدة أجهزة** (كاشير + جهاز لوحي + مكتب): فعّل Supabase — الخطوات في ملف **`دليل-قاعدة-البيانات.md`** وملف **`supabase-schema.sql`** المرفقين

---

## 🚀 للتشغيل الدائم في النادي (نسخة الإنتاج)

عند جاهزية النظام للعمل اليومي، ابنِ نسخة محسّنة:
```
npm run build
```
ينشأ مجلد `dist` يحتوي النظام جاهزاً. لتشغيله:
```
npm run preview
```
أو ارفع مجلد `dist` على أي استضافة مجانية مثل **Netlify** أو **Vercel** (اسحب المجلد فقط) لتحصل على رابط دائم يفتحه أي جهاز في النادي من أي مكان.

---

## أسئلة سريعة

**"npm غير معرّف" أو "command not found"؟**
Node.js لم يُثبّت بعد أو يحتاج إعادة فتح موجّه الأوامر — أعد الخطوة 1 وأعد فتح النافذة.

**تعطّل عند `npm install`؟**
تحتاج اتصال إنترنت لهذه الخطوة (لتحميل مكتبات React مرة واحدة). بعدها يعمل النظام دون إنترنت (عدا المزامنة السحابية وتوليد PDF).

**كيف أوقف النظام؟**
في نافذة موجّه الأوامر اضغط `Ctrl + C`.

**كيف أضع شعار النادي واسمه؟**
بعد الدخول: الإعدادات → المظهر (الشعار) والكلمات والمعلومات (الاسم والعنوان).
